Compute class gradients :param net [nn.Module]: neural network :param to_eval [torch.Tensor]: what to evaluate :param R [torch.Tensor]: adjency matrix :param norm_exponent [int] = 2: norm exponent for computing the score

Formula:
d/dyi P_theta(yj|x) = d/dyi max{ P_theta(y_j | x), P_theta (y_j, x) }

= d/dyi (z P_theta (y_j | x) + (1 - z) P_theta(y_j | x) ) [z in { 0, 1 }] =~ d/dP_theta(yi | x) (z P_theta (y_j | x) + (1 - z) P_theta(y_j | x) ) [z in { 0, 1 }] = z

foreach i,j in {1…#classes}
