chmncc.utils.utils.resume_training(resume_training: bool, experiment_name: str, model: Module, optimizer: Optimizer) → Tuple[Dict, Dict, int][source]

Resumes the training if the corresponding flag is specified. If the resume_training flag is set to true, the function tries to recover, from the checkpoint specified, the number of epoch, training and validation parameters. If the resume_training flag is set to false, the parameters are set to the default ones

Parameters:

  • [nn.Module] (model) – network architecture

  • [str] (experiment_name) – where the weight file is located

  • [torch.optim.Optimizer] (optimizer) – optimizer

Returns:

training parameters val_params [Dict]: validation parameters [step, best_loss] start_epoch [int]: start epoch number

Return type:

training_params [Dict]
